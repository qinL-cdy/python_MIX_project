## 查看项目
在网页中点击进入对应文件即可
<br/>
## 上传项目
进入对应文件夹点击Upload files  
根据提示上传文件即可
<br/>
## 修改项目
点击右上角fork  
这将拷贝一份项目到你自己的仓库  
在自己的仓库中点击Clone or download  
将项目克隆到本地  
根据你的想法修改对应的代码  
提交你的修改到自己的仓库    
在自己仓库的项目中点击pull request  
原项目管理员将审核你的请求并合并到项目中
